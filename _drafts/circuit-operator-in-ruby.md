Is used as an idiom:

{% highlight ruby %}
@user ||= "the test"
{% endhighlight %}

is equivalent to

{% highlight ruby %}
@user = @user || "the test"
{% endhighlight %}

This is just a shortcut, a very handy one, if you want to assign new values to a variable if it still hasn't got one
